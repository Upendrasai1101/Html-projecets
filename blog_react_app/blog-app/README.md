# Advanced React Blog Application

React.js mariyu Material UI upayoginchi build chesina oka complete Blogging Platform. Indulo Advanced React Hooks mariyu State Management concepts cover cheyabaddayi.

## 🚀 Key Features
- **User Authentication**: AuthContext mariyu Context API upayoginchi Global Login/Logout state handle cheyabaddindi.
- **Protected Routes**: Login ayina users matrame posts create chesela React Router tho security implement cheyabaddindi.
- **State Management**: Complex logic kosam `useReducer` mariyu form handling kosam `useRef` upayoginchadam jarigindi.
- **Custom Hooks**: API calls kosam `useFetch` mariyu persistent data kosam `useLocalStorage` hooks create cheyabaddayi.
- **API Integration**: Axios upayoginchi external REST API nundi data fetching mariyu CRUD operations implement cheyabaddayi.

## 🛠️ Tech Stack
- **Frontend**: React.js, Material UI (MUI)
- **Routing**: React Router v6
- **State**: Context API, useReducer
