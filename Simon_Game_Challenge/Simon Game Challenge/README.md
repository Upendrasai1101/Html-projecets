# Simon Game - Interactive Memory Challenge

A classic memory-testing game built with Vanilla JavaScript, jQuery, and CSS. The game generates a random sequence of button flashes and sounds, challenging the player to replicate the pattern.

## 🚀 Game Features
- **Dynamic Level Scaling**: The sequence grows longer and faster as the player progresses.
- **Audio Feedback**: Unique sounds for each button to provide an immersive experience.
- **Visual Effects**: Custom CSS animations and shadows for button presses and game-over states.
- **Responsive Logic**: Handled user input sequences and compared them against generated patterns in real-time.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3
- **Scripting**: JavaScript (ES6)
- **Library**: jQuery (for DOM manipulation and event handling)

## 🎮 How to Play
1. Open the `index.html` file in any modern web browser.
2. Press any key on your keyboard to start the game.
3. Watch the sequence shown by the game (buttons will flash and make sounds).
4. Click the buttons in the same order.
5. If you miss a sequence, it's Game Over! Press any key to restart.
