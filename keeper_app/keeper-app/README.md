# Keeper App (Google Keep Clone)

React.js mariyu Material UI upayoginchi create chesina oka dynamic note-taking application.

## 🚀 Features
- **Add Notes**: Title mariyu content tho kotha notes ni ventane create cheyochu.
- **Delete Notes**: Avasaram leni notes ni click thone delete cheyochu.
- **Interactive UI**: Material UI Icons, Fab (Floating Action Button), mariyu Zoom animations upayoginchadam jarigindi.
- **Conditional Rendering**: Input area expansion mariyu hover effects neat ga handle cheyabaddayi.

## 🛠️ Tech Stack
- **Frontend**: React.js
- **Styling**: Material UI & CSS3
- **Hooks**: useState (State Management kosam)
